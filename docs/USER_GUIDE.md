# User Guide

- **대시보드**: 운영 현황과 주의 항목을 확인합니다.
- **프로젝트**: 고객사·프로젝트·리소스로 검색하고 상태로 필터링합니다. 행을 선택하면 고유 상세 URL로 이동합니다.
- **일정**: 연도와 확률을 선택해 1~12월 작업 기간을 봅니다. 노란 열은 현재 월입니다.
- **리소스**: 월을 선택해 배정률을 봅니다. 100% 초과는 과부하, 80~100%는 용량 임박입니다.
- **CSV**: 프로젝트 화면의 내보내기는 한국어 Excel 호환 BOM을 포함합니다.

표시된 2026 일정은 원본 스크린샷의 월 단위 잠정 정보입니다. 상세 일정을 확정할 때 실제 날짜와 할당률을 수정하십시오.
