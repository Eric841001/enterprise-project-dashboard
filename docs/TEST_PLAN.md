# Test Plan

자동 테스트는 필수값·범위·날짜 검증, 불연속 일정, 과부하 계산, 상태/확률 경고, UTF-8 CSV를 다룹니다.

수동 인수 테스트:

- 이메일 로그인, 세션 유지, 로그아웃, 비밀번호 재설정
- Admin/Manager의 고객·리소스·프로젝트 생성·편집·보관
- Viewer에서 모든 변경 동작 차단 및 직접 API 쓰기 거부
- 여러 리소스와 할당률 저장, 월별 100% 초과 경고
- 1~12월 일정, 불연속 기간, 프로젝트 링크 확인
- CSV 한국어 내보내기와 Excel 열기
- 휴대폰 메뉴·표·타임라인 사용성
- 새로고침과 직접 프로젝트 URL 접근
- Pages Actions의 typecheck, lint, test, build, deploy 성공

삭제·복원, CSV 가져오기, 감사 이력은 후속 CRUD 구현 시 테스트를 추가합니다.
